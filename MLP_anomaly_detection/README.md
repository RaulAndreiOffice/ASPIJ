# MLP_anomaly_detection
Is a Multilayer Perceptron for anomaly detection in fittnes




<img width="837" height="297" alt="image" src="https://github.com/user-attachments/assets/28126c66-4d39-453c-be55-7111308842d2" />


During preprocessing, we removed Id, Calories Burn, and Actual Weight from the dataset.

We established:
Input: ~20 neurons

Hidden layers: 64 → 32, all with ReLU activation

Output: 1 neuron (predicted pulse), linear activation
